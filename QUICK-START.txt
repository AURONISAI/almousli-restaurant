================================================
  DEPLOYMENT STEPS
================================================

1. Extract this folder on your server

2. Install dependencies:
   npm install --production

3. Start the server:
   npm start

The website will run on PORT 3000

================================================
  FOR ONE.COM HOSTING
================================================

1. Upload via FTP to your web directory
2. Extract the zip file
3. In cPanel > Node.js Application:
   - App Root: /path/to/this/folder
   - Startup File: node_modules/next/dist/bin/next
   - Arguments: start
4. Click 'Restart'

================================================
  TROUBLESHOOTING
================================================

Site won't start?
- Check Node.js version (must be 18+)
- Run: npm install --production

Port already in use?
- Edit package.json, change port in start script
- Or use: PORT=8080 npm start
