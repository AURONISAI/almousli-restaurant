================================================================
    GITHUB + VERCEL DEPLOYMENT GUIDE
================================================================

This guide will help you deploy the Almousli Restaurant website
to Vercel using GitHub.

================================================================
    STEP 1: PUSH TO GITHUB
================================================================

The code is already committed locally. Now create a GitHub repo:

1. Go to https://github.com/new
2. Create a new repository:
   - Repository name: almousli-restaurant
   - Description: Damascus Syrian Restaurant Website
   - Visibility: Private (recommended) or Public
   - Do NOT initialize with README (we already have one)

3. After creating, GitHub will show you commands. Use these:

cd c:\Users\samij\Desktop\ALmousli
git remote add origin https://github.com/YOUR_USERNAME/almousli-restaurant.git
git branch -M main
git push -u origin main

Replace YOUR_USERNAME with your GitHub username.

================================================================
    STEP 2: DEPLOY TO VERCEL
================================================================

Option A: Using Vercel Dashboard (Easiest)
-------------------------------------------

1. Go to https://vercel.com
2. Sign in with GitHub (or create account)
3. Click "Add New..." > "Project"
4. Select "Import Git Repository"
5. Find "almousli-restaurant" in the list
6. Click "Import"
7. Configure project:
   - Framework Preset: Next.js (auto-detected)
   - Root Directory: ./
   - Build Command: npm run build (auto-detected)
   - Output Directory: .next (auto-detected)
8. Add Environment Variables:
   - NEXT_PUBLIC_PHONE=0737330513
   - NEXT_PUBLIC_SITE_URL=https://your-domain.com
   - (Add others as needed)
9. Click "Deploy"
10. Wait 2-3 minutes for deployment to complete
11. Your site is LIVE! âœ¨

Option B: Using Vercel CLI
---------------------------

1. Install Vercel CLI:
   npm i -g vercel

2. Login:
   vercel login

3. Link project:
   cd c:\Users\samij\Desktop\ALmousli
   vercel link

4. Deploy:
   vercel --prod

================================================================
    STEP 3: CONFIGURE ENVIRONMENT VARIABLES
================================================================

In Vercel Dashboard:

1. Go to your project
2. Click "Settings" tab
3. Click "Environment Variables"
4. Add each variable:

   Required:
   ---------
   NEXT_PUBLIC_PHONE=0737330513
   NEXT_PUBLIC_SITE_URL=https://almousli.se

   Optional:
   ---------
   NEXT_PUBLIC_WHATSAPP=46737330513
   NEXT_PUBLIC_EMAIL=info@almousli.se
   NEXT_PUBLIC_HERO_VIDEO=/videos/hero-background.mp4

   Sanity CMS (if using):
   ----------------------
   NEXT_PUBLIC_SANITY_PROJECT_ID=your_project_id
   NEXT_PUBLIC_SANITY_DATASET=production
   SANITY_API_TOKEN=your_token

5. Select which environments: Production, Preview, Development
6. Click "Save"

================================================================
    STEP 4: CONFIGURE CUSTOM DOMAIN
================================================================

In Vercel Dashboard:

1. Go to your project
2. Click "Settings" > "Domains"
3. Add your domain: almousli.se
4. Vercel will provide DNS configuration:

   Option A: Vercel Nameservers (Recommended)
   -------------------------------------------
   Update at your domain registrar:
   ns1.vercel-dns.com
   ns2.vercel-dns.com

   Option B: A/CNAME Records
   -------------------------
   Add to your DNS:
   Type: A
   Name: @
   Value: 76.76.21.21

   Type: CNAME
   Name: www
   Value: cname.vercel-dns.com

5. Wait 24-48 hours for DNS propagation
6. SSL certificate is automatic!

================================================================
    TROUBLESHOOTING
================================================================

Build Failed:
- Check build logs in Vercel dashboard
- Verify all environment variables set
- Test locally: npm run build

Images Not Loading:
- Verify public/ folder committed to Git
- Check next.config.mjs image domains

Site Slow:
- Enable Vercel Analytics
- Check Web Vitals
- Optimize images
- Review bundle size

Domain Not Working:
- Wait 24-48 hours for DNS propagation
- Use nslookup to check DNS
- Verify DNS records correct
- Check Vercel domain settings
