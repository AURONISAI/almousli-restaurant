================================================================
    STEP 1: UPLOAD TO ONE.COM
================================================================

1. Login to your One.com control panel
2. Go to "File Manager" or use FTP client
3. Upload "almousli-deployment.zip" to your web directory
4. Extract the ZIP file on the server

================================================================
    STEP 2: INSTALL DEPENDENCIES
================================================================

Via SSH (if available):
    cd /path/to/your/folder
    npm install --production

Or use One.com's terminal/command line interface

================================================================
    STEP 3: CONFIGURE NODE.JS APP
================================================================

In One.com Control Panel:

1. Go to: "Advanced" > "Node.js"
2. Click "Add Node.js Application"
3. Set the following:
   
   Application Root: /path/to/almousli-deployment
   Application Startup File: node_modules/next/dist/bin/next
   Application Arguments: start
   Node.js Version: 18.x or higher (LTS recommended)
   Port: 3000 (default)

4. Click "Create" or "Save"

================================================================
    STEP 4: CONFIGURE DOMAIN
================================================================

1. In One.com panel, go to "Domains"
2. Point your domain to the Node.js application
3. Set up SSL certificate (usually automatic)

================================================================
    STEP 5: START THE APPLICATION
================================================================

1. In Node.js Applications panel, click "Restart"
2. Wait 10-30 seconds for startup
3. Visit your domain to verify

================================================================
    ENVIRONMENT VARIABLES (Optional)
================================================================

If you need to set custom environment variables:

1. In Node.js app settings, look for "Environment Variables"
2. Add these (or edit .env.local file):

   NEXT_PUBLIC_PHONE=0737330513
   NEXT_PUBLIC_SITE_URL=https://your-domain.com
   NEXT_PUBLIC_HERO_VIDEO=/videos/hero-background.mp4

================================================================
    TROUBLESHOOTING
================================================================

Application won't start:
  - Check Node.js version (must be 18+)
  - Run: npm install --production
  - Check application logs in One.com panel

Port conflict:
  - Change port in One.com Node.js settings
  - Or modify package.json start script

Images not loading:
  - Verify public/ folder uploaded completely
  - Check file permissions (644 for files, 755 for folders)

Slow performance:
  - Enable Gzip compression (usually automatic)
  - Check server resources in One.com panel
  - Monitor using Node.js logs
